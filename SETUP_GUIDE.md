# 🧱 E-COMMERCE PROJECT SETUP GUIDE

## 📌 Overview

## ⚙️ 1. System Requirements

### **Software**
- **XAMPP** or **WAMP** (local server environment)
- **PHP** 8.0 or later
- **MySQL** 8.0 or later
- **VS Code** (or preferred code editor)
- **Git** (for version control)

---
